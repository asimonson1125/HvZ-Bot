## Install
Clone repository:  
```git clone https://github.com/asimonson1125/HvZ-Bot```

Install dependencies:  
```npm install```

Create `auth.js` with template:
```js
export const token = "BOT_TOKEN";
export const SQLUser = "USER";
export const SQLPass = "PASSWORD";
```

All done!  
Run: `node index.js`
